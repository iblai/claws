Common compliance frameworks to be aware of:
- SOX (Sarbanes-Oxley): financial reporting and internal controls
- HIPAA: health information privacy and security
- GDPR: EU data protection and privacy
- SOC 2: service organization controls for data security
- PCI DSS: payment card data security
- OSHA: workplace safety and health standards

Certification renewal tracking should start 90 days before expiration.
Training completion records must be retained per regulatory requirements (varies by framework).
