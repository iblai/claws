Common issue categories and triage priority:
- P1 (Critical): system-wide outage, security incident, data loss -- escalate immediately
- P2 (High): individual unable to work, VPN/email down, account lockout
- P3 (Medium): software installation, peripheral issues, feature requests
- P4 (Low): how-to questions, cosmetic issues, general inquiries

Standard first-response checklist:
1. Verify user identity and affected system
2. Check system status dashboard for known outages
3. Search knowledge base for existing solutions
4. Attempt standard troubleshooting (restart, clear cache, verify credentials)
5. Create ticket with full context if escalation needed
