# Data Sources

Systems and platforms commonly accessed for meeting management and follow-up.

## Calendar & Scheduling

- **Google Calendar** -- meeting details (event ID, title, description, start time, end time, time zone, recurrence rule, status, visibility, transparency), attendee lists (email, display name, RSVP status, organizer flag, optional flag, comment, response date), recurring meetings (recurrence pattern, exceptions, instance dates, modifications, cancellations), room bookings (room resource, building, floor, capacity, AV equipment, booking status, check-in status), event descriptions (agenda, dial-in info, attachments, links, meeting notes link, preparation materials)
- **Microsoft Outlook Calendar** -- meeting invites (meeting ID, subject, body, start/end time, time zone, location, categories, importance, sensitivity, show as), attendee tracking (required attendees, optional attendees, resources, response status, response time, delegate responses), room reservations (room name, building, floor, capacity, equipment, availability, booking policy, auto-accept), scheduling assistant (free/busy data, suggested times, room availability, conflict detection, working hours, time zone resolution)
- **Calendly** -- meeting scheduling (event type, duration, availability rules, buffer time, minimum notice, booking page URL, questions), availability (connected calendars, working hours, date overrides, round-robin availability, timezone handling), round-robin (team members, distribution method, priority, availability, booking count, equalization), team pages (team URL, members, event types, routing rules, admin settings), routing forms (form ID, questions, conditional logic, routing destination, booking outcome, conversion rate)
- **Doodle** -- group scheduling (poll ID, title, options, participants, votes, final selection, deadline, comments), meeting coordination (time slots, participant availability, overlap analysis, suggested time, notification, confirmation)

## Video Conferencing & Recording

- **Zoom** -- meeting recordings (meeting ID, recording ID, file type, file size, download URL, play URL, recording start/end, passcode, sharing settings, storage location), transcripts (meeting ID, transcript text, speakers, timestamps, paragraphs, word-level timing, language, confidence), cloud storage (recording ID, file size, duration, resolution, views, downloads, retention date, auto-delete), attendance reports (meeting ID, participant name, email, join time, leave time, duration, attentiveness score, in-waiting-room time), chat logs (meeting ID, sender, receiver, message, timestamp, file shared)
- **Microsoft Teams** -- meeting recordings (meeting ID, recording URL, organizer, start/end time, size, expiration date, SharePoint/OneDrive location, sharing permissions), transcripts (meeting ID, transcript content, speakers, timestamps, confidence, language, download URL, editing history), attendance (meeting ID, participant, join time, leave time, duration, role, attention signal, device), chat history (meeting ID, sender, content, timestamp, reactions, mentions, files, importance, deleted/edited)
- **Google Meet** -- meeting recordings (meeting ID, recording URL, organizer, start/end time, duration, storage location, access permissions, expiration), attendance tracking (meeting code, participant email, join time, leave time, duration, calling from, device type), transcripts (meeting ID, transcript text, speakers, timestamps, language, live caption status, accuracy, Google Workspace edition)
- **Webex** -- meeting recordings (meeting ID, recording URL, host, start/end time, duration, format, size, password, expiration, sharing link), transcripts (meeting ID, transcript text, speakers, timestamps, highlights, action items, auto-chaptering), attendee reports (meeting ID, participant, email, join/leave time, duration, role, attention tracking, audio/video status), space messaging (space ID, messages, files, mentions, threads, integrations, whiteboard snapshots)

- **Fireflies.ai** -- meeting transcription (transcript ID, meeting title, date, participants, transcript text, speakers, timestamps, duration, channel), AI summaries (summary, key points, action items, decisions, questions, follow-ups, outline, keywords), action items (action ID, text, assignee, mentioned by, timestamp, meeting, status, due date), CRM integration (meeting ID, CRM record, contact, account, deal, logged activity, notes sync), topic tracking (topic, frequency, meetings, speakers, time spent, sentiment, trend)
- **Fathom** -- meeting recording (recording ID, meeting, platform, duration, participants, video URL, highlights, clips), AI highlights (highlight ID, timestamp, type, speaker, text, importance, category, clip URL), action items (action ID, text, assignee, context, timestamp, meeting, follow-up status), CRM sync (meeting, CRM platform, synced record, contact, deal, activity type, notes, custom fields)
- **Avoma** -- conversation intelligence (meeting ID, recording, transcript, speakers, talk ratio, questions asked, monologue length, patience, engagement), meeting notes (AI-generated notes, topics, key moments, decisions, action items, custom templates, manual edits), coaching (rep, meeting scores, talk patterns, question frequency, objection handling, next steps quality, peer comparison), revenue intelligence (deal, meeting count, stakeholder engagement, buying signals, competitor mentions, pricing discussion, sentiment trend)
- **Gong** -- conversation recording (call ID, participants, duration, recording URL, platform, deal, account, contact, call type), deal intelligence (deal ID, engagement score, stakeholder map, activity gap, risk signals, competitor mentions, next steps, win probability), coaching insights (rep, talk-to-listen ratio, longest monologue, question rate, filler word frequency, topic coverage, patience, interactivity), talk patterns (avg talk time, response latency, interruption count, engaging questions, next steps mentioned, pricing discussion, follow-up commitment)
