# Data Sources

Systems and platforms commonly accessed for training content development.

## Learning Management & Delivery

- **Cornerstone OnDemand** -- course catalog (course ID, title, description, format, duration, provider, version, status, skill tags), SCORM/xAPI hosting (content package ID, standard version, launch URL, tracking data, completion criteria), learner analytics (enrollment count, completion rate, average score, time-on-course, drop-off point, satisfaction rating), compliance tracking (regulation, jurisdiction, training requirement, assignment rules, expiration, audit log)
- **SAP SuccessFactors Learning** -- curriculum management (curriculum ID, courses, sequence, prerequisites, version, approval status, target audience), certification programs (cert ID, required courses, passing criteria, validity period, renewal requirements, issuing authority), regulatory training (regulation code, jurisdiction, training modules, completion evidence, audit trail, update frequency)
- **Workday Learning** -- content delivery (content ID, format, hosting location, launch method, mobile compatibility, accessibility compliance), learning campaigns (campaign ID, target population, content, delivery schedule, reminder cadence, completion metrics), blended learning programs (program ID, components, sequence, ILT sessions, digital modules, assessment checkpoints)
- **Docebo** -- content marketplace (content ID, provider, topic, format, rating, license type, usage rights), AI content curation (recommendation engine data, content relevance scores, user engagement predictions, personalization rules), learning impact analytics (course effectiveness score, knowledge retention rate, behavior change indicators, business impact correlation)

## Authoring Tools

- **Articulate Storyline / Rise** -- interactive e-learning modules (project ID, slides/blocks, interactions, media assets, variables, triggers, layers), branching scenarios (decision points, paths, feedback rules, scoring logic, completion criteria), responsive design (breakpoints, layout rules, responsive behavior, mobile optimization, accessibility features)
- **Adobe Captivate** -- simulation-based learning (simulation type, screen captures, click boxes, text entry, branching, scoring), VR content (360-degree scenes, hotspots, interactions, navigation, device compatibility), responsive design (fluid boxes, breakpoints, device preview, gesture support, theme templates)
- **Lectora (ELB Learning)** -- SCORM/xAPI compliant modules (packaging standard, manifest, SCO structure, sequencing rules, communication data), accessibility compliance (Section 508 conformance, WCAG level, alt text, keyboard navigation, screen reader compatibility, color contrast ratios)
- **iSpring Suite** -- PowerPoint-to-e-learning conversion (source presentation, conversion settings, output format, player customization, LMS compatibility), quizzes (question types, grading rules, feedback, branching, attempt limits, passing score), dialog simulations (scenario tree, character selection, response options, scoring, feedback messages)
- **Camtasia (TechSmith)** -- screen recordings (resolution, frame rate, cursor effects, annotations, callouts, zoom/pan), video tutorials (clips, transitions, annotations, captions, audio tracks, interactive hotspots), interactive quizzes (question overlay timing, question type, feedback, SCORM reporting, results tracking)

## Content Libraries & OER

- **LinkedIn Learning** -- pre-built professional development courses (course ID, title, author, skill tags, level, duration, language, exercise files, transcript), skill paths (path ID, courses, estimated hours, skill outcomes, completion badge)
- **Skillsoft Percipio** -- curated content channels (channel ID, topic, content count, format mix, skill alignment, freshness date), practice labs (lab ID, technology, environment specifications, duration, objectives, hands-on exercises), audiobooks and books (title, author, ISBN, summary, skill tags, duration/pages, chapters)
- **OpenSesame** -- e-learning content marketplace (course ID, provider, topic, format, duration, languages, compliance focus, SCORM version), curated catalogs (catalog ID, theme, course count, curation criteria, update frequency)
- **GO1** -- content aggregation (content sources, content types, topic coverage, language support, update frequency), compliance libraries (regulation, jurisdiction, course modules, version, last updated, certification), professional development (skill area, course catalog, learning paths, estimated hours, completion credentials)

## Assessment & Survey

- **Questionmark** -- assessment authoring (assessment ID, item bank, question types, randomization rules, scoring algorithm, psychometric properties), psychometric analysis (item difficulty, discrimination index, reliability coefficient, validity measures, distractor analysis), proctoring (proctoring type, identity verification, session recording, violation flags, incident reports), certification exams (exam ID, blueprint, cut score, form assembly rules, passing rate, item statistics)
- **SurveyMonkey / Qualtrics** -- training needs analysis surveys (survey ID, questions, response scales, target audience, distribution method, response rate), post-training evaluations (Kirkpatrick Level 1 reaction data, Level 2 knowledge gain scores, confidence ratings, open-ended feedback, NPS), course feedback (rating questions, likert scales, comment themes, sentiment analysis, improvement suggestions)
- **Kahoot!** -- gamified quizzes (quiz ID, questions, time limits, point values, answer options, media attachments), interactive presentations (slide content, poll questions, word clouds, brainstorm prompts), engagement tools (participation rate, accuracy rate, response time, leaderboard standings, engagement score)

## Media & Asset Management

- **Panopto** -- video hosting (video ID, title, duration, resolution, storage size, folder, permissions, retention policy), lecture capture (recording source, camera feed, screen capture, presentation slides, chapter markers), in-video quizzing (quiz ID, insertion point, question type, correct answer, completion requirement), analytics (views, unique viewers, average watch time, drop-off points, search queries within video)
- **Kaltura** -- video management (entry ID, title, description, tags, categories, access control, thumbnail, flavors/transcodes), interactive video (chapters, hotspots, quizzes, paths, annotation layers), live streaming (event ID, stream URL, DVR, recording, chat, polls, Q&A), LMS integration (LTI configuration, grade passback, content embedding, SSO, analytics sync)
- **Canva / Adobe Creative Cloud** -- graphic design (design ID, template, dimensions, brand assets, color palette, typography, export format), infographics (data visualization elements, icons, layout, text blocks, source data), visual assets (image library, illustrations, stock photos, custom graphics, brand-compliant templates)
- **DAM systems (Bynder, Brandfolder)** -- digital asset management (asset ID, file type, dimensions, tags, metadata, usage rights, expiration), brand templates (template ID, brand guidelines, customizable fields, approved variations, version history), media libraries (folder structure, asset categories, search taxonomy, access permissions, download tracking)

## Standards & Compliance

- **SCORM Cloud (Rustici)** -- content testing (content package, SCORM version, test results, error log, compatibility report), LRS functionality (statements received, statement format, query API, data export, retention), cross-platform compatibility verification (LMS targets, test matrix, pass/fail results, issue descriptions, remediation notes)
- **xAPI / cmi5** -- learning activity tracking (actor, verb, object, result, context, timestamp, authority, attachments), competency-based assessment data (competency framework, competency ID, mastery threshold, evidence, assertion), experience data (activity type, duration, score, completion, success, response, extensions)
- **Section 508 / WCAG tools (WAVE, axe)** -- accessibility compliance testing (WCAG level target, violations found, violation type, element reference, remediation guidance, impact severity), automated scan results (page URL, issues by category, contrast ratios, missing alt text, ARIA errors, keyboard traps)
