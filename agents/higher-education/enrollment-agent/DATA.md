# Data Sources

Systems and platforms commonly accessed for enrollment and admissions workflows.

## Student Information Systems (SIS)

- **Ellucian Banner** -- student demographics (name, DOB, SSN-last-four, gender, ethnicity, citizenship), enrollment status (full-time/part-time, term, level), admit type and entry term, academic program and major codes, contact information (address, phone, email), residency classification, housing intent flag, prior institution records, SEVIS data for international students
- **Ellucian Colleague** -- student bio-demographic profiles (name, DOB, address, race/ethnicity), admissions application records (application date, admit status, decision date), enrollment history by term (credits attempted, credits earned, cumulative GPA), prospect/applicant/student lifecycle stage, communication plan logs
- **Oracle PeopleSoft Campus Solutions** -- prospect records (source, territory, recruiter assignment), admissions application data (app ID, program, plan, admit type, decision status), enrollment management snapshots (headcount, FTE, census data), student group assignments, academic career and program stack, external test scores, prior education records
- **Workday Student** -- student lifecycle events (inquiry, application, admission, enrollment, graduation), admissions pipeline stage and probability, enrollment analytics (yield rates, melt rates, funnel conversion), demographic and diversity data, program of interest, net tuition revenue projections
- **Jenzabar One** -- student master records (ID, demographics, contact info), admissions tracking (application status, checklist items, decision codes), enrollment funnel data (inquiries, apps, admits, deposits, enrolled), activity/attribute codes, student flags and tags

## CRM & Recruitment Platforms

- **Technolutions Slate** -- prospect records (source, inquiry date, territory, counselor assignment), inquiry tracking (form submissions, event RSVPs, portal activity), application status (started, submitted, complete, decision rendered), event registrations (campus visits, open houses, admitted student days), communication history (emails sent/opened/clicked, SMS, print), reader scorecards and committee decisions, travel schedules, territory assignments, drip campaign membership, portal login activity, interview scheduling, financial aid estimate submissions
- **Salesforce Education Cloud** -- lead records (source, status, score, owner), recruitment campaign data (campaign members, responses, ROI), prospect engagement tracking (email opens, web visits, event attendance), opportunity pipeline (stage, probability, expected value), contact records (student, parent/guardian, counselor), task and activity logs, report and dashboard data, custom object records for applications and decisions
- **Ellucian CRM Recruit** -- prospect pipeline stages (suspect, prospect, inquiry, applicant, admit, enrolled), territory management (geographic, program-based assignments), recruitment event data (registrations, attendance, follow-up), communication plan execution (templates, schedules, delivery status), recruiter activity metrics (contacts made, events attended, territory coverage), predictive enrollment scoring
- **HubSpot** -- email campaign analytics (sends, opens, clicks, bounces, unsubscribes), landing page submissions (form fields, UTM parameters, conversion rates), lead scoring (engagement score, fit score, lifecycle stage), contact properties (source, first touch, last activity), workflow enrollment and completion data, blog/content engagement metrics
- **TargetX** -- recruitment workflow stages (inquiry through enrollment), event management (registration, attendance, follow-up tasks), communication plans (email, SMS, print sequences), application portal activity, counselor territory data, retention alerts, student success milestones

## Application & Admissions

- **Common Application (CommonApp)** -- application submissions (personal information, citizenship, demographics, family, education, testing, activities, honors, essays), recommendation letters (counselor, teacher, other), school report (GPA, class rank, curriculum, school profile), mid-year and final reports, fee waiver status, application fee payments, FERPA release status, Common App ID
- **Coalition Application** -- application data (personal information, education history, activities, essays), portfolio/locker submissions (documents, media, reflections), collaboration history, recommender assignments and submissions
- **SUNY Application** -- state university system application data (personal info, residency, program selection, prior education, test scores, essay), supplemental materials, EOP eligibility indicators
- **ApplyTexas** -- Texas institution application data (personal info, residency determination, academic history, major selection, essays by topic), dual-credit and concurrent enrollment history, auto-admit indicators (top 6% / top 10% rule)
- **Technolutions Slate (Reader)** -- application review queues (reader assignments, queue depth, completion rates), reader scorecards (rubric criteria, individual scores, comments), committee review decisions (admit, deny, waitlist, defer), holistic review notes, rating distributions, inter-rater reliability metrics
- **Parchment** -- transcript delivery records (sender institution, recipient, delivery date, status), document verification (transcript authenticity, seal verification), electronic transcript data (courses, grades, GPA, graduation date), credential status tracking

## Communication & Engagement

- **Mongoose Cadence** -- SMS/text messaging campaigns (message templates, send volumes, delivery rates), two-way texting conversations (student responses, sentiment, opt-out rates), campaign analytics (response rates, engagement by segment), contact preferences, conversation history, staff assignment and routing
- **EAB Navigate** -- student outreach campaigns (target populations, messaging, delivery), appointment scheduling (advisor, date, type, status, notes), engagement tracking (campaign responses, appointment attendance, action item completion), early alert data, student success markers
- **Twilio** -- SMS delivery logs (to, from, status, timestamp, error codes), voice call records (duration, disposition), communication API usage metrics, opt-in/opt-out status, delivery receipts
- **Mailchimp / Constant Contact** -- email campaign performance (sends, opens, clicks, bounces, unsubscribes by campaign), drip sequence enrollment and progression, list segmentation data, A/B test results, engagement scoring over time, subscriber profile data, audience growth metrics

## Testing & Standardized Data

- **College Board** -- SAT scores (total, ERW, math, essay if applicable, test date, percentiles), AP scores (subject, score 1-5, year), CSS Profile data (parent income, assets, family size, number in college, home equity, business/farm value, non-custodial parent data), SAT score-send history
- **ACT** -- ACT scores (composite, English, math, reading, science, writing if applicable, test date, state testing vs. national), enrollment planning data (EPS responses: intended major, housing plans, financial aid interest, college preferences)
- **TOEFL / IELTS** -- TOEFL iBT scores (total, reading, listening, speaking, writing, test date, MyBest scores), IELTS scores (overall band, listening, reading, writing, speaking, test date, TRF number), score validity dates
- **National Student Clearinghouse** -- enrollment verification (institution, enrollment status, dates, level), prior enrollment history (all institutions attended, dates, degree earned), degree verification (degree type, major, date conferred), transfer-out and stop-out indicators

## Analytics & Reporting

- **EAB Enroll360** -- enrollment predictive models (likelihood to enroll, yield probability, net revenue estimates), market analytics (competitor overlap, market share, geographic penetration), financial aid optimization (discount rate modeling, merit aid scenarios), funnel conversion benchmarks
- **Rapid Insight Veera** -- predictive enrollment analytics (enrollment probability scores, key predictor variables, model accuracy metrics), scenario modeling outputs, segmented yield predictions, financial aid leveraging recommendations
- **Tableau / Power BI** -- enrollment dashboards (funnel metrics by term/program/territory, conversion rates at each stage), yield visualization (admitted-to-enrolled conversion, melt tracking), demographic breakdowns (by geography, ethnicity, gender, program), historical trend comparisons, census snapshot reports
