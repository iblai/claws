Common issue categories and first-response steps:
- Password/account lockout: verify identity, initiate reset through approved workflow
- Wi-Fi connectivity: check device settings, verify credentials, check for outages
- LMS access: verify enrollment status, check browser compatibility, clear cache
- Email issues: check webmail access, verify forwarding rules, check storage quota
- VPN: verify client version, check credentials, confirm network allows VPN traffic
- Software installation: check license availability, verify system requirements

Always verify the user's identity before making account changes.
