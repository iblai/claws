# Data Sources

Systems and platforms commonly accessed for faculty support and course administration.

## Learning Management Systems (LMS)

- **Canvas (Instructure)** -- course content management (modules, pages, files, external URLs, content licensing, publication status), gradebook (assignment groups, weighting schemes, grading schemes, grade posting policies, late submission policies, grade history), assignments (title, description, points, due date, availability window, submission type, rubric, peer review settings, group assignments, Turnitin integration), quizzes (question types: multiple choice, essay, matching, fill-in-the-blank; question banks, time limits, attempt limits, quiz statistics), SpeedGrader (inline annotation, rubric scoring, audio/video feedback, grade assignment, submission comments), rubrics (criteria, rating descriptions, point values, outcome alignments, scoring method: points/percentage/free-form), Modules (module structure, prerequisite modules, completion requirements, sequential progression settings), announcements (title, message, posted date, delayed posting, notification preferences)
- **Blackboard (Anthology)** -- course sites (content areas, course menu structure, adaptive release rules, availability settings), grade center (grade columns, categories, weighted totals, grading schemas, calculated columns, inline grading), content areas (folders, items, file attachments, web links, mashups, learning modules, lesson plans), assessment tools (tests, surveys, question pools, random blocks, test options, grade feedback), SafeAssign plagiarism detection (originality reports, similarity scores, matched sources, institutional database matching, global database matching)
- **Brightspace (D2L)** -- course builder (content modules, topic types: files, links, HTML, SCORM; release conditions, date restrictions, completion tracking), Intelligent Agents (trigger criteria: login activity, grade thresholds, content access; automated email notifications, agent run history), rubrics (analytic and holistic rubrics, criteria, levels of achievement, competency alignment), portfolio (student artifacts, reflections, assessment items, presentation pages), competency tracking (competency frameworks, learning objectives, achievement levels, evidence mapping)
- **Moodle** -- course resources (files, folders, pages, URLs, books, labels, IMS content packages), quiz engine (question types, question banks, random question selection, adaptive mode, grading method, review options), workshop activity (peer assessment assignments, submission phase, assessment phase, grading evaluation phase, allocation methods), grade book (grade items, categories, aggregation methods, scales, outcomes, grade letters), turnitin integration (assignment settings, similarity reports, GradeMark, PeerMark)

## Curriculum & Catalog Management

- **CourseLeaf (Leepfrog Technologies)** -- course proposal workflows (new course proposals, course modification requests, course deactivation, workflow steps, approver assignments, approval status, comments, effective catalog year), catalog publishing (course descriptions, prerequisites, co-requisites, credit hours, grading mode, repeat rules, cross-listing, offered terms, GenEd designations), curriculum approval process (program proposals, major/minor changes, degree requirement modifications, committee review stages), program modification (program learning outcomes, requirement changes, credit hour adjustments, effective date)
- **Curriculog (Anthology)** -- curriculum proposal management (proposal type: new course, course change, new program, program change; originator, current step, approvers, decision history), approval workflows (workflow templates, step sequence, parallel/serial routing, voting thresholds, comment threads, revision requests), impact reports (prerequisite impact, enrollment impact, resource impact, cross-listing implications, affected programs)
- **Acalog (Anthology)** -- academic catalog publishing (programs, degrees, majors, minors, certificates, general education requirements), program requirements (course requirements, elective groups, credit minimums, GPA requirements, milestone requirements, core/concentration/elective structures), course descriptions (title, description, credits, prerequisites, co-requisites, fees, offered terms, cross-listing, equivalent courses)

## Academic Integrity

- **Turnitin** -- plagiarism detection (Similarity Report: overall similarity percentage, color-coded match breakdown, matched sources list), similarity reports (source matches by percentage, internet sources, publications, student paper repository matches, excluded sources), GradeMark feedback (inline comments, QuickMark rubric stamps, voice comments, text comments, summary comments), originality scoring (similarity index, internet match percentage, publication match percentage, student paper match percentage)
- **Copyleaks** -- AI content detection (AI probability score, human-written probability, mixed content identification, sentence-level analysis), plagiarism checking (similarity score, matched sources, side-by-side comparison, excluded quotations/bibliography), code plagiarism (code similarity detection across programming languages, structural comparison, variable renaming detection)
- **Respondus LockDown Browser / Monitor** -- secure testing (browser lockdown settings, blocked functions: copy, paste, print, screen capture; allowed URLs, calculator access), proctoring (webcam video recording, facial detection, environment scan, flagged events: missing student, multiple people, phone detected), webcam monitoring (session recordings, flag severity levels, review queue, instructor review decisions)

## Assessment & Survey Tools

- **ExamSoft (Anthology)** -- secure exam delivery (exam scheduling, download windows, upload deadlines, exam duration, question randomization, answer scrambling), item analysis (item difficulty index, discrimination index, point-biserial correlation, distractor analysis), psychometric data (exam reliability: Cronbach's alpha, KR-20; score distribution, mean, median, standard deviation), competency mapping (exam items mapped to competencies/learning outcomes, student performance by competency, program-level assessment aggregation)
- **Scantron** -- optical mark recognition (scan results: student ID, answer selections, score), test scoring (raw scores, scaled scores, percentile ranks, item analysis), item analysis (item difficulty, discrimination, distractor effectiveness, reliability coefficients)
- **Qualtrics** -- course evaluations (response rates, Likert scale ratings by question, open-ended comments, demographic breakdowns), student surveys (custom survey instruments, branching logic, response data, completion rates), research instruments (survey design, distribution, response collection, data export, statistical analysis)
- **EvaluationKIT (Watermark)** -- course evaluation management (evaluation periods, question sets, response collection, reminder schedules), faculty reporting (individual evaluation reports, item means, response distributions, longitudinal comparisons, comment summaries), benchmarking (department averages, college averages, institutional averages, peer comparisons by course type/level)
- **IDEA (Center for Evaluation & Educational Policy)** -- student ratings (teaching effectiveness ratings, progress on relevant objectives, course difficulty, student effort, student motivation), diagnostic feedback (strengths and areas for improvement, teaching method effectiveness by objective type), teaching improvement (comparative data, recommended strategies, longitudinal tracking of evaluation scores)

## Research & Scholarly Tools

- **ORCID** -- researcher identifiers (unique ORCID iD, linked publications, grant associations, employment affiliations, education history), publication tracking (works list, DOIs, metadata, source attribution, visibility settings), grant associations (funded projects, funder, grant number, project dates, role)
- **Google Scholar** -- citation indexes (publications list, citation counts per publication, total citations, h-index, i10-index), publication search (full-text search, author profiles, co-author networks), h-index tracking (career h-index, recent h-index, h-index trend)
- **Scopus / Web of Science** -- citation databases (publication records, citation counts, citing articles, field-weighted citation impact), impact metrics (journal impact factor, CiteScore, SJR, SNIP, article-level metrics), collaboration networks (co-author analysis, institutional collaboration maps, international collaboration indicators)
- **Zotero / Mendeley / EndNote** -- reference management (reference libraries, citation metadata: author, title, journal, year, DOI, volume, pages; tags, notes), bibliography generation (citation style formatting: APA, MLA, Chicago, etc.; in-text citations, reference lists, footnotes), library organization (collections, tags, search, duplicate detection, PDF storage, annotation)

## Faculty Information & Workload

- **Digital Measures / Watermark Faculty Success** -- faculty activity reporting (teaching: courses taught, credit hours, student evaluations; scholarship: publications, presentations, grants, creative works; service: committees, advising, community engagement), CV management (automated CV generation, section formatting, data import from publication databases, multiple CV templates), workload documentation (credit hour assignments, course releases, administrative duties, overload compensation, reassigned time), tenure and promotion dossiers (narrative statements, evidence of teaching/scholarship/service, external review letters, supporting documentation, committee evaluations)
- **Interfolio** -- faculty search (position postings, application collection, committee access, applicant evaluation forms, EEO data, shortlist management), review and appointment management (review case assembly, file sharing, committee deliberation tools, voting, external reviewer solicitation), tenure and promotion workflows (case creation, document assembly, committee review stages, provost/dean review, decision recording, notification)
- **Banner HR / Workday HCM** -- faculty contracts (appointment type: tenure-track, tenured, non-tenure-track, adjunct, visiting; rank, salary, contract dates, FTE), course load (assigned courses, credit hours taught, overload status, independent study supervision, dissertation/thesis committee membership), compensation (base salary, stipends, summer pay, overload pay, administrative supplements), leave tracking (sabbatical applications, FMLA, sick leave, vacation, professional development leave)

## Classroom & Scheduling

- **Ad Astra** -- course scheduling (course sections, room assignments, meeting patterns: days, times, duration; instructor assignments, section caps), room assignments (room capacity, room type: lecture, seminar, lab, studio; equipment inventory, accessibility features), space utilization analytics (room utilization rates, seat fill rates, peak usage times, underutilized spaces, scheduling efficiency metrics)
- **EMS (Event Management System)** -- classroom reservations (room booking: date, time, duration, event type, requestor, confirmation status), event scheduling (setup/teardown times, recurring events, resource requirements, catering, AV needs, conflicts)
- **Crestron / Extron** -- classroom AV control (projector/display control, audio routing, camera control, source switching, room presets), lecture capture integration (recording start/stop triggers, scheduled recordings, AV routing to capture systems, room status monitoring)
- **Panopto / Kaltura / YuJa** -- lecture recording (recording schedule, duration, quality settings, auto-publish settings, storage usage), video management (video library, folder organization, sharing permissions, embed options, captions/transcripts), student viewing analytics (views per video, average watch time, drop-off points, engagement heatmaps, unique viewers, viewing completion rates)
