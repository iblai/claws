# Data Sources

Systems and platforms commonly accessed for tutoring and academic support.

## Learning Management Systems (LMS)

- **Canvas (Instructure)** -- course syllabi (learning objectives, weekly topics, required readings, instructor policies), assignment details (title, description, due date, points possible, submission type, rubric criteria), rubrics (criteria names, rating descriptions, point values, outcome alignments), due dates (assignment calendar, upcoming deadlines, late policy), grade data (current scores, assignment group weights, running grade, submission feedback, graded rubric results), module content (module sequence, content items, completion requirements, prerequisite modules)
- **Blackboard (Anthology)** -- course materials (content areas, folders, learning modules, file attachments), gradebook (grade columns, item scores, weighted grades, grading schemas, submission attempts), assessment details (test questions, question types, point values, pool sources, availability windows), content areas (organized learning materials, SCORM packages, publisher integrations)
- **Brightspace (D2L)** -- course content (content modules, topic pages, HTML content, embedded media, release conditions), quizzes (question types, point values, attempts allowed, time limits, correct answers for review), discussion prompts (topic descriptions, grading criteria, participation requirements), learning outcomes (competency frameworks, outcome alignments, achievement scales)
- **Moodle** -- course resources (files, URLs, pages, books, labels), assignment specifications (description, submission type, file requirements, word limits, group assignments), quiz banks (question categories, question types, randomization settings, answer keys), forum discussions (discussion types, initial posts, reply threads, grading rubrics)

## Tutoring & Academic Support Platforms

- **Penji** -- tutoring session scheduling (available subjects, tutor availability windows, session duration, session location/modality), subject matching (tutor expertise areas, student course enrollment, skill level pairing), session notes (topics covered, student comprehension assessment, recommended follow-up, resource links shared), attendance records, session ratings and feedback
- **TutorMe (GoGuardian)** -- on-demand tutoring (session request subject, wait time, tutor match, session start/end timestamps), session history (subjects tutored, session duration, tutor ratings, concepts covered, whiteboard recordings), subject coverage (available subjects/courses, tutor specialization areas, coverage gaps), student satisfaction ratings
- **Tutor.com (Princeton Review)** -- session transcripts (chat logs, whiteboard content, shared documents, session recordings), resource recommendations (suggested readings, practice problems, video links), usage analytics (sessions per student, average duration, peak usage times, subject distribution, satisfaction scores), drop-in vs. scheduled session data
- **CRLA (College Reading & Learning Association)** -- tutor certification standards (Level 1/2/3 requirements, training hours completed, topics covered), training resources (tutor training modules, best practices, assessment techniques, learning theory foundations), program accreditation status
- **Knack** -- peer tutoring marketplace (tutor profiles, verified course grades, availability, hourly rates), session scheduling (booking calendar, confirmation status, cancellation tracking), tutor matching (course-specific matching, student preferences, tutor ratings), session feedback and outcomes

## Course Content & Open Educational Resources (OER)

- **OpenStax** -- open textbooks for introductory courses: subject catalogs (biology, chemistry, physics, math, economics, psychology, history, sociology, etc.), chapter structure (sections, key terms, worked examples, practice problems, critical thinking questions), ancillary resources (instructor solutions, test banks, lecture slides, simulations)
- **MIT OpenCourseWare** -- lecture notes (topic outlines, key concepts, derivations, worked examples), problem sets (problem statements, difficulty level, solution sets), exams (past exams, practice exams, solution keys), course structure (prerequisite chains, recommended background, reading lists)
- **Khan Academy** -- instructional videos (topic, subtopic, duration, skill level, transcript), practice exercises (question banks, mastery-based progression, hints, worked solutions), course-aligned content (subject mapping to common university courses, skill prerequisites, mastery tracking), progress data (skills mastered, skills in progress, energy points)
- **Wolfram Alpha** -- computational knowledge (step-by-step math solutions, formula references, unit conversions), math problem solving (algebra, calculus, linear algebra, differential equations, statistics with solution steps), data lookups (scientific constants, chemical properties, statistical datasets, geographic data)
- **Desmos** -- graphing calculator (function plotting, table of values, sliders for parameter exploration, regression tools), geometry tools (construction tools, transformations, measurements, proofs), interactive math activities (teacher-created investigations, pre-built explorations, assessment items)

## Assessment & Practice

- **McGraw Hill Connect** -- homework platforms (assigned problems by chapter/topic, adaptive learning paths, due dates, attempt limits), adaptive learning (SmartBook reading assignments, knowledge probes, confidence-based assessment, mastery metrics), practice problems (question banks by topic, difficulty levels, hints, solution walkthroughs), student performance data (time on task, completion rates, mastery levels, struggling topics)
- **Pearson MyLab** -- course-specific practice (homework exercises, quiz prep, study plans by chapter), tutorials (step-by-step guided solutions, help-me-solve-this walkthroughs, video explanations, animation-based instruction), assessments (auto-graded homework, quizzes, tests, personalized study plans based on performance), learning catalytics data (real-time student response data, concept-check results)
- **Cengage MindTap** -- interactive assignments (reading activities, application exercises, case studies, embedded media), readings (digital textbook chapters, highlights, annotations, shared notes), practice tests (chapter review quizzes, exam prep, flashcards, learning path recommendations), student engagement metrics (reading time, activity completion, performance trends)
- **WebAssign** -- online homework for STEM courses (physics, math, chemistry, statistics, engineering), problem banks (textbook-aligned problems, algorithmic variations, randomized values, multi-part problems), student response data (answer submissions, attempt counts, time spent, partial credit), tutorial support (Watch It videos, Read It textbook links, Practice Another Version), grade passback to LMS
- **Chegg Study (reference only)** -- solution approaches for common textbook problems (step-by-step solutions, expert Q&A, textbook solution manuals), problem-solving methodology reference (not for direct answer provision)

## Student Records

- **SIS (Banner, PeopleSoft, Workday)** -- enrolled courses (current term schedule, section details, instructor, meeting times, credits), prerequisites completed (course history relevant to current enrollment, prerequisite chain validation), academic standing (current standing, GPA, credit hours completed, class level, Dean's List/probation status), advisor information, registration holds
- **EAB Navigate / Starfish** -- tutoring referrals (referring faculty/staff, course context, specific areas of concern, urgency level), early alerts (alert type, academic or attendance concern, course section, date raised, recommended intervention), academic risk indicators (risk score, contributing factors, trend direction, prior interventions and outcomes)
