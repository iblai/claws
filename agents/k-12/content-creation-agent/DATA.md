# Data Sources

Systems and platforms commonly accessed for K-12 content creation.

## Content & Resource Platforms

- **Teachers Pay Teachers** -- educator-created resource marketplace
  - **Resources**: title, resource_type (worksheet/activity/assessment), grade_range, subject, standards, file_format
  - **Metadata**: ratings, reviews, preview_pages, seller_info, price

- **Canva for Education** -- visual design platform for educators
  - **Templates**: presentation_templates, worksheet_templates, infographic_templates, poster_templates
  - **Assets**: images, icons, illustrations, brand_kit, class_assignments

- **Google Slides / PowerPoint** -- presentation tools
  - **Content**: slides, speaker_notes, embedded_media, animations, templates

- **Newsela** -- leveled nonfiction content
  - **Articles**: original_text, leveled_versions (multiple lexile levels), quiz_questions, annotations, writing_prompts
  - **Text sets**: theme, curated_articles, standards_alignment, grade_range

- **ReadWorks** -- reading comprehension resources
  - **Passages**: text, lexile_level, grade_range, subject, question_sets, vocabulary, paired_texts

- **PBS LearningMedia** -- multimedia educational content
  - **Resources**: type (video/interactive/lesson_plan/image), grade, subject, duration, standards

## Curriculum Platforms

- **Discovery Education** -- digital curriculum and media
  - **Content**: video_segments, interactives, virtual_labs, reading_passages, image_library
  - **Channels**: subject, grade, topic, standards_alignment, related_resources

- **BrainPOP** -- animated educational content
  - **Topics**: animated_movie, quiz, activity, vocabulary, graphic_organizer, game
  - **Alignment**: standards, grade_suitability, subject, related_topics

- **Nearpod** -- interactive lesson platform
  - **Activities**: slides, polls, open_ended_questions, draw_it, collaborate_boards, VR_field_trips
  - **Reports**: participation_rate, responses, completion_status

## Accessibility & Differentiation

- **Bookshare (Benetech)** -- accessible textbooks for students with print disabilities
  - **Formats**: DAISY, BRF (braille), audio, large_print, word_highlight_sync

- **UDL Guidelines (CAST)** -- Universal Design for Learning framework
  - **Checkpoints**: engagement, representation, action_expression, grade_appropriate_examples
