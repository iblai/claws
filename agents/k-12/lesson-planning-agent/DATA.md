# Data Sources

Systems and platforms commonly accessed for K-12 lesson planning.

## Standards Databases

- **Common Core State Standards (CCSS)** -- math and ELA standards for K-12
  - **Standards**: standard_id, domain, cluster, grade_level, full_text, mathematical_practice
  - **Progressions**: learning_progression across grades, prerequisite_standards, related_standards

- **Next Generation Science Standards (NGSS)** -- science standards
  - **Performance expectations**: PE_code, description, grade_band, disciplinary_core_idea
  - **Three dimensions**: science_practice, crosscutting_concept, core_idea
  - **Bundled arrangements**: topic, performance_expectations, connections_to_CCSS

- **State standards portals** -- state-specific learning standards
  - **Standards**: state, subject, grade, standard_code, description, strand, benchmark
  - **Pacing guides**: unit, week, standards_covered, suggested_duration

## Curriculum & Content Platforms

- **OpenStax** -- free K-12 aligned textbooks
  - **Content**: chapter, section, learning_objectives, key_terms, practice_problems, teacher_guide

- **BetterLesson** -- teacher-created lesson plans and strategies
  - **Lessons**: title, grade, subject, standards_aligned, strategy_type, student_work_samples

- **Teachers Pay Teachers** -- educator marketplace
  - **Resources**: resource_type, grade_range, subject, standards, preview, ratings, reviews

- **Newsela** -- leveled nonfiction content
  - **Articles**: title, topic, lexile_levels (multiple readings), quiz_questions, writing_prompts, standards_alignment

- **PBS LearningMedia** -- multimedia educational resources
  - **Media**: type (video/interactive/document), grade_range, subject, standards, duration

## Learning Management Systems (LMS)

- **Canvas / Schoology / Google Classroom** -- lesson delivery platforms
  - **Course structure**: modules, assignments, pages, quizzes, discussions, rubrics
  - **Calendar**: due_dates, scheduled_content, pacing
  - **Student data**: prior_performance_by_standard, completion_rates, class_averages

## Assessment Data (for informing lesson design)

- **NWEA MAP / iReady / Renaissance Star** -- diagnostic data
  - **Class profiles**: class_average_RIT, distribution_by_instructional_level, growth_areas
  - **Grouping data**: students_by_readiness_band, suggested_instructional_focus, skill_gaps
