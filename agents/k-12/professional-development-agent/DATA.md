# Data Sources

Systems and platforms commonly accessed for K-12 professional development.

## PD Management & Learning

- **Frontline Professional Growth** -- PD tracking and management
  - **Catalog**: course_name, provider, format (in-person/online/hybrid), credit_hours, cost, dates
  - **Transcripts**: teacher_id, courses_completed, credits_earned, certificate_status, renewal_date
  - **Plans**: individual_professional_development_plan, goals, activities, completion_status

- **Alludo** -- gamified PD platform for educators
  - **Badges**: badge_name, competency_area, activities_completed, evidence_submitted
  - **Progress**: learning_path, modules_completed, micro-credentials_earned

- **Coursera for Campus / edX** -- online professional learning
  - **Courses**: course_name, provider, duration, completion_certificate, skill_tags
  - **Progress**: modules_completed, quiz_scores, peer_assessments, time_invested

## Observation & Evaluation

- **Danielson Framework for Teaching** -- teacher evaluation framework
  - **Domains**: planning_and_preparation, classroom_environment, instruction, professional_responsibilities
  - **Components**: 22 components with rubric levels (unsatisfactory, basic, proficient, distinguished)

- **Marzano Focused Teacher Evaluation Model** -- alternative evaluation framework
  - **Domains**: standards-based_planning, standards-based_instruction, conditions_for_learning, professional_responsibilities
  - **Elements**: 23 elements with scale levels (not_using, beginning, developing, applying, innovating)

- **TeachBoost / Whetstone** -- observation and coaching platforms
  - **Observations**: observer, date, look-fors, evidence_notes, rubric_scores, action_steps
  - **Coaching cycles**: pre-conference, observation, post-conference, follow-up, goal_progress

- **Swivl / Edthena** -- video-based coaching
  - **Videos**: classroom_recording, timestamp_markers, self-reflection_notes, coach_comments
  - **Analytics**: talk_time_ratio, questioning_frequency, student_engagement_markers

## Student Data (for data-driven PD)

- **SIS / LMS / Assessment platforms** -- student performance data
  - **Class data**: assessment_results, grade_distributions, growth_measures, standard_mastery_by_class
  - **Trends**: year_over_year_comparison, subgroup_performance, DFW_rates

## State Certification & Licensure

- **State educator portals** -- licensure and renewal tracking
  - **Certification**: certificate_type, endorsement_areas, issue_date, expiration_date, renewal_requirements
  - **CEU tracking**: professional_development_hours, approved_providers, credit_categories
